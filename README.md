# sugarcoat

#### 介绍
框架选型:
springboot
undertow
lombok
hutool工具类
权限框架：springSecurity
文档框架：knife4j spring-doc
持久化：mybatis-plus
日志框架:logback
定时任务：quartz

"email	    邮箱功能"
"enums	    枚举类封装"
"exception	异常处理"
"log		日志组件"
"monitor	监控组件"

"sugarcoatParam	    参数配置"
"protection	系统保护组件"
"sms		短信封装"
"status	    启停状态封装"
"tenant	    动态多租户组件"
"validate	校验组件"
